Squat King is a Jump King pseudo-mod made with Python and Mediapipe.
The program uses camera-based fullbody tracking to detect certain landmarks
and calculate poses to determine game inputs.

As the name implies, you play by Squatting!
(No need to go to the gym anymore, trust me)


Note: This video is out of date (based on v1.0), refer to the changelog for the latest updates.

Squat King is still in development and I plan to add and improve functionalities,
so if you have any suggestion you can leave it here or DM me on twitter.
Thank you! ^^

Note: Windows only!
